=encoding utf8

=head1 Name

Mo::is - Adds the is feature to Mo's has

=head1 Synopsis

    use Mo qw'is';
    has name => ( is => 'ro' );

=head1 Description

Adds the is parameter to has. If you set it to I<ro>, the accessor
will only work as a getter. You will still able to set the value 
at construction time via C<new>.

=cut
