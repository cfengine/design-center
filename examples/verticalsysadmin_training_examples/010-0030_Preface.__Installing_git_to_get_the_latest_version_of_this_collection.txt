Installing git to download the latest version of these examples
---------------------------------------------------------------

The examples in this collection are kept on GitHub.  Install
the git client to download the latest version of these examples.

Installing Git for RHEL 5 systems:

    rpm -ihv  http://dl.fedoraproject.org/pub/epel/5/i386/epel-release-5-4.noarch.rpm 

    yum install git
