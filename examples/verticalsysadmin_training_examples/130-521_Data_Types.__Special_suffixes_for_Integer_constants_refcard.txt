Integer Suffixes
----------------

    'k' = value times 1000.
    'm' = value times 1000^2
    'g' = value times 1000^3

    'K' = value times 1024.
    'M' = value times 1024^2
    'G' = value times 1024^3

    '%' meaning percent, in limited contexts

    'inf' = a constant representing an unlimited value.
